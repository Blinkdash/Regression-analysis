Выполнен регрессионный анализ датасета цен ноутбуков. Проведены описательный анализ и предобработка данных, исследованы распределения и корреляции признаков. Построены базовые регрессионные модели (Constant, ConstantByGroup, OneParamModel), линейные модели с учетом только колличественных и всех признаков (с нормализацией и без). Реализованы полиномиальная регрессия, дерево решений и случайный лес, выполнено сравнение моделей по метрикам MSE и R² и анализ важности признаков.
